# kakao-assignment-1

React와 Tailwind CSS로 구현한 Todo 관리 앱입니다. 할 일을 추가하고, 수정하고, 완료 상태를 변경하거나 삭제할 수 있으며, 필터 탭을 통해 원하는 상태의 Todo만 확인할 수 있습니다.

## 주요 기능

- 할 일 추가
  - 입력창에 할 일을 작성한 뒤 `추가` 버튼을 클릭하거나 Enter 키로 Todo를 추가할 수 있습니다.
  - 빈 값을 입력하면 안내 메시지를 표시합니다.

- 할 일 수정
  - 각 Todo의 `수정` 버튼을 누르면 항목 내부에 인라인 입력창이 표시됩니다.
  - 수정 중 `저장` 버튼으로 내용을 반영하고, `취소` 버튼으로 수정 상태를 종료할 수 있습니다.
  - 빈 값으로 저장하려고 하면 안내 메시지를 표시합니다.

- 완료 상태 변경
  - `완료` 버튼을 누르면 Todo가 완료 상태로 바뀌고 취소선이 표시됩니다.
  - 완료된 Todo는 `취소` 버튼으로 다시 진행중 상태로 되돌릴 수 있습니다.

- 할 일 삭제
  - `삭제` 버튼을 눌러 Todo를 목록에서 제거할 수 있습니다.

- Todo 필터링
  - `전체`, `진행중`, `완료` 3가지 필터 탭을 제공합니다.
  - 선택한 탭에 따라 Todo 목록이 필터링됩니다.
  - 현재 선택된 탭은 배경색과 그림자로 시각적으로 구분됩니다.
  - 필터별로 Todo가 없을 때 다른 빈 상태 문구를 표시합니다.

- 날짜 선택 및 필터링
  - `DateNavigator` 컴포넌트로 날짜를 이동할 수 있습니다.
  - `이전`, `다음` 버튼으로 날짜를 하루씩 변경합니다.
  - 선택된 날짜(예: `2026년 6월 10일`)가 화면에 표시됩니다.
  - 선택된 날짜의 Todo만 목록에 표시됩니다.
  - 새 Todo 추가 시 현재 선택된 날짜가 자동으로 저장됩니다.
  - 날짜 필터와 상태 필터(`all/active/completed`)가 함께 적용됩니다.

- 데이터 자동 저장
  - 모든 Todo 데이터는 브라우저의 `localStorage`에 JSON 형식으로 자동 저장됩니다.
  - Todo를 추가, 수정, 완료, 삭제할 때마다 즉시 저장됩니다.
  - 페이지를 새로고침해도 이전 Todo 목록과 상태가 복원됩니다.
  - 저장되는 정보: `id`, `text`, `completed`, `date` (YYYY-MM-DD 형식)

```txt
    TodoFilter.jsx
    TodoForm.jsx

```

## 컴포넌트 역할

- `App.jsx`
  - Todo 목록, 안내 메시지, 현재 필터 상태, 선택된 날짜를 관리합니다.
  - Todo 추가, 수정, 완료 토글, 삭제 로직을 담당합니다.
  - 현재 필터와 선택된 날짜에 맞는 Todo 목록을 계산해 `TodoList`에 전달합니다.
  - `useEffect`를 사용해 Todo 변경 시 `localStorage`에 자동으로 데이터를 저장합니다.
  - 앱 시작 시 `localStorage`에서 저장된 Todo 목록을 불러옵니다.

- `DateNavigator.jsx`
  - 날짜 선택 UI를 담당합니다.
  - `이전`, `다음` 버튼으로 날짜를 하루씩 변경합니다.
  - 현재 선택된 날짜를 `YYYY년 MM월 DD일` 형식으로 표시합니다.
  - `onChangeDate` 콜백을 통해 선택된 날짜 상태를 App에 전달합니다.

- `TodoForm.jsx`
  - 새 Todo를 입력하고 추가하는 폼입니다.
  - 입력된 텍스트와 현재 선택된 날짜를 함께 제출합니다.

- `TodoFilter.jsx`
  - `전체`, `진행중`, `완료` 필터 탭을 렌더링합니다.
  - 현재 선택된 필터를 시각적으로 강조합니다.

- `TodoList.jsx`
  - Todo 목록을 렌더링합니다.
  - 목록이 비어 있을 때 필터 상태에 맞는 빈 상태 문구를 표시합니다.

- `TodoItem.jsx`
  - 개별 Todo 항목을 렌더링합니다.
  - 수정, 완료 토글, 삭제 UI를 제공합니다.

## 기술 스택

- React 18.3.1
- Tailwind CSS 3.4.17
- Vite 5.4.11

## 주요 구현 사항

### 날짜 기반 Todo 관리
- 각 Todo에 `YYYY-MM-DD` 형식의 날짜 정보를 저장합니다.
- `DateNavigator` 컴포넌트를 통해 날짜를 변경하고 선택된 날짜의 Todo만 표시합니다.
- 새 Todo 추가 시 자동으로 현재 선택된 날짜가 저장됩니다.

### localStorage 자동 저장
- `useEffect` 훅을 사용해 `todos` 배열이 변경될 때마다 자동으로 저장됩니다.
- 저장된 데이터는 JSON 형식으로 `todos` 키에 저장됩니다.
- 앱 시작 시 `loadTodos()` 함수가 저장된 데이터를 불러와 초기 상태로 설정합니다.

### 복합 필터링
- 날짜 필터와 상태 필터(`all/active/completed`)가 함께 동작합니다.
- 사용자는 특정 날짜의 진행중 또는 완료된 Todo만 필터링할 수 있습니다.

## 실행 방법
